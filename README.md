# parakeet-ptt

Push-to-talk voice dictation for Claude Code (and any terminal) powered by NVIDIA's [Parakeet TDT 0.6B v3](https://huggingface.co/nvidia/parakeet-tdt-0.6b-v3) — a state-of-the-art open-source ASR model running fully locally on your GPU.

Hold **F9** → speak → release → transcribed text is pasted directly into your focused window.

---

## Why

Typing long prompts into Claude Code is slow. Parakeet TDT runs at ~3,000× real-time on a mid-range GPU, so a 10-second recording transcribes in under a second. Everything stays local — no API calls, no audio leaving your machine.

## Requirements

- **OS**: Ubuntu 20.04+ (or Debian-based Linux) with an X11 session
- **GPU**: NVIDIA GPU with 4 GB+ VRAM
- **Driver**: NVIDIA driver 520+ (CUDA 12.1 compatible)
- **Disk**: ~6 GB for the virtualenv and model weights

## Install

```bash
git clone <repo>
cd parakeet
bash install.sh
```

The script handles everything:
- Installs Python 3.10, `xclip`, `xdotool`, and audio libraries via `apt`
- Auto-detects your CUDA version from the installed driver
- Creates a virtualenv at `parakeet/env/` (stays off your OS drive)
- Downloads the Parakeet TDT 0.6B v3 weights into `parakeet/model_cache/` (~2.4 GB)
- Generates `start_listener.sh`

At the end it prints the exact path to use for starting the listener.

## Usage

Start the listener once before you need it:

```bash
bash start_listener.sh
```

You'll see a live counter while the model loads (~20 seconds), then:

```
[parakeet] Ready — hold F9 to record.
```

From that point on, anywhere in X11:

| Action | Result |
|---|---|
| Hold **F9** | Mic starts recording — notification appears |
| Release **F9** | Audio is transcribed and pasted into the focused window |

Transcription appears as a desktop notification and lands directly in whatever window was active when you pressed F9 — even if you switch focus during the ~1–4 second transcription.

## Customising the vocabulary

Parakeet is a general ASR model — it transcribes what it hears phonetically. `corrections.py` lets you map spoken forms to the written forms you actually want:

```python
# corrections.py
CORRECTIONS = [
    (r'\bgemma\s+(?:4|four|for)\s*b\b',  'gemma:4b'),
    (r'\bllama\s+(?:3|three)\b',          'llama3'),
    (r'\bdeep[\s-]seek\b',               'deepseek'),
    # add your own...
]
```

Each entry is a `(regex_pattern, replacement)` pair applied case-insensitively. Changes take effect on the next recording — no restart needed.

## Changing the push-to-talk key

Edit `PTT_KEY` near the top of `listener.py`:

```python
PTT_KEY = "f9"        # default
PTT_KEY = "f10"       # alternative
PTT_KEY = "scroll_lock"
PTT_KEY = "pause"
```

Must be a [pynput `keyboard.Key`](https://pynput.readthedocs.io/en/latest/keyboard.html#pynput.keyboard.Key) name — not a regular character key, or it will also type into your terminal.

## Changing the paste method

Find the `── Paste shortcut ──` section in `listener.py` and uncomment the line for your terminal:

```python
# Ctrl+Shift+V  — gnome-terminal, xterm, alacritty, kitty (default)
# Ctrl+V        — VS Code integrated terminal
# Shift+Insert  — universal X11 fallback
# xdotool type  — types directly, no clipboard
```

For Wayland, switch the clipboard block from `xclip` to `wl-copy`.

## How it works

```
F9 held
  └─ sounddevice captures mic at 16 kHz
F9 released
  └─ audio saved to temp WAV
      └─ NeMo loads Parakeet TDT on GPU
          └─ corrections.py post-processes text
              └─ xclip copies to clipboard
                  └─ xdotool focuses original window + pastes
```

The listener runs as a background daemon. Logs go to `/tmp/parakeet_listener.log`.

```bash
# Check status
ps -p $(cat /tmp/parakeet_listener.pid)

# View live log
tail -f /tmp/parakeet_listener.log

# Stop
kill $(cat /tmp/parakeet_listener.pid)
```

## Why NeMo and not Hugging Face Transformers?

The official `nvidia/parakeet-tdt-0.6b-v3` weights ship in NeMo's native checkpoint format. The TDT decoder — which gives accurate word-level timestamps and handles noisy/long-form audio better than CTC — is not yet in the stable Transformers release. NeMo is the only way to run it with official weights.

If you want to avoid NeMo, swap in `nvidia/parakeet-ctc-0.6b` with standard `transformers` — accuracy is comparable for clean audio, and no NeMo install is needed. See `INSTALL.md` for the swap-in instructions.
